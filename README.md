# 🚀 Power BI 作品集：快速啟動指南

歡迎光臨！這份指南將帶您快速啟動並載入本專案的 Power BI 報表模型 (`.pbit`)。
透過範本檔 (.pbit) 與資料範例 (.csv) 的分離，您可以輕鬆地在自己的電腦上重現報表，甚至替換成自己的資料！

---

## 📥 1. 下載 .pbit 資料分析模型格式

**什麼是 .pbit？**
`.pbit` (Power BI Template) 是 Power BI 的「範本檔」。它包含了報表精心設計的所有視覺效果、精美的 UI 版面、資料模型架構，以及複雜的 DAX 量值（也就是**除了實際資料以外**的所有精華）。

> [!tip] 點擊下載您的專案範本：
> 📂 **[`我的作品.pbit`]** (請確認檔案已保存到您的電腦中)

---

## 📊 2. 下載 .csv 資料範例

報表需要吃資料才能產生出美麗的圖表！為了讓模型順利跑起來，我們準備了一份完整的示範用原始資料集。

> [!tip] 點擊下載並解壓縮原始資料：
> 📂 **[`Sales.csv.zip`]** 👉 解壓縮後可得到 **`Sales.csv.csv`** 檔案。
> 🚨 *請務必記住您稍後解壓縮存放 `Sales.csv.csv` 的資料夾路徑喔！*

---

## 🛠️ 3. 使用方式 (啟動與連線)

這三個簡單的步驟可以讓您的範本瞬間充滿資料：

### 步驟 A：開啟範本
對著剛剛下載的 💛 **`我的作品.pbit`** 連按兩下開啟，系統會自動啟動 Power BI Desktop。

### 步驟 B：資料來源提示視窗
在啟動且進入資料轉換階段的瞬間，畫面畫面中央會跳出一個**「連接資料來源」**的參數提示視窗：

> [!info] 💡 提示畫面示意
> 系統會問您一個問題：「請輸入資料來源的路徑」
> 
> ```
> ┌──────────────────────────────────────────────┐
> │ 輸入參數                                        │
> ├──────────────────────────────────────────────┤
> │                                              │
> │ 參數1 "Sales_Data_Path":                     │
> │ [ 請在這裡貼上 Sales.csv.csv 的絕對路徑...     ] │
> │                                              │
> │                                       [載入] │
> └──────────────────────────────────────────────┘
> ```

### 步驟 C：填入檔案路徑並載入
請直接在原本空白的輸入框中，貼上您在【步驟 2】解壓縮出來的 `Sales.csv.csv` 的**完整檔案路徑**。

> **✅ 路徑範例：**
> `C:\Users\YourName\Downloads\Sales.csv.csv`

最後，勇敢地按下 **「載入 (Load)」**！
喝口水稍待片刻，看著 Power BI 為您重新整理資料... 登愣！🎉 完整的互動式分析儀表板就會完美呈現在您的眼前了！
